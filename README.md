# Yêu cầu
 - Cài đặt dotnet-cli (Optional), 
1. Clone repo 
```
git clone https://github.com/vudinhan2525/SE310-FoodApp.git
```
2. Cd vào folder backend
```
cd .\SE310-FoodApp\backend\
```
3. Chạy server nếu đã cài dotnet cli hoặc chạy bằng visual studio
```
dotnet watch
```
